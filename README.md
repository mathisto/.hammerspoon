# Setup
1. Install Hammerspoon: `brew cask install hammerspoon`
1. Go to your 'home' directory: `cd ~`
1. Clone the repo: `git clone git@github.com:mathisto/.hammerspoon.git`

# Features
- Auto-reloading config
- Use of F19 key as 'Hyper' key
- Single key Hyper chords
- Cmd-Hyper chords
